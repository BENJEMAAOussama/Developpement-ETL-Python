Developpement ETL avec Python:
Le but est d'extraire de données depuis un compte Spotify (Titres et chanteurs des chansons ecoutées ...), à l'aide de Spotify API.
Ensuite de selectionner les données pertinantes (les chansons écoutées que durant les dernieres 24 heures) et finalement de les stockées dans une base de données sqlite.
